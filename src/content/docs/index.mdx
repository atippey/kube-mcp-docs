`kubemcp.io` helps you run Model Context Protocol servers as first-class Kubernetes resources.

## What you can do quickly

- Deploy an `MCPServer` and expose it through cluster ingress.
- Register multiple `MCPTool` resources behind one server.
- Publish reusable `MCPPrompt` templates and `MCPResource` endpoints.
- Use Kubernetes RBAC, NetworkPolicy, and metrics from day one.
